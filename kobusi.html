This is HakureiChat 2! A simple SNS developed by only a person with App Inventor 2.

What we have…

  • No personal information collection
  • No IP address collection
  • No Ads
  • No personalized contents
  • Compatibility with HakureiChat 1
  • Multilingual UI (Currently work in progress, some of strings may still remain in English, will be fixed soon!)

Please note that this software is still in beta! You may join the beta-testing program now by joining the Google Space and download the .APK file in the File section!
